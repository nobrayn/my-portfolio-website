DESIGN TIPS

TEXT WIDTH
wrap full-screen width text in a container and set a max width:
.wrapper {
  max-width: 60rem
  margin: 0 auto;
}

Try the ch unit?  60 characters per line would be:
  max-width: 60ch;


BREATHING ROOM
.wrapper {
  padding: 0 2em // if no padding, then reduce width
  width: 100%
}

Adding gap in flex container: //  gap may not be supported
.flexContainer {
  display: flex;
  gap: 1em;
}

Got a background color box? Give it padding and margin. Top and bottom here, as left right margin is taken care of by wrapper:
padding: 5rem 0;
margin: 5rem 0;

Consider changing line height above the default:
body {
  line-height: 1.5;
}

However, line height should be reduced for headings
h1, h2, h3 {
  line-height: 1.1;
}


CONTRAST
Consider darker headings than paragraphs, etc.


TEXT-ALIGNMENT
Avoid justify text (text-align: justify;)
Limit text-align: center; unless it's on short pieces of text

CONTRAST WITH FONT PROPERTIES
font-weight - bold for headings, not for... the rest
font-size in rem - may have to reduce the ratio in media queries for smaller screens:

:root {
  --fs-400: 1rem
  --fs-400: 1.125rem
  --fs-400: 1.375rem
  --fs-400: 2rem

}
@media (min-width: 45em) {
  --fs-400: 1.125rem
  --fs-500: 1.375rem
  --fs-600: 2.75rem
  --fs-700: 4.125rem
}

Notes for above - this is for min-width.. flip it for the max-width that you know and love:

/* Portrait tablet and small desktops */
@media (max-width: 940px) {
}

/* Landscape phone to portrait tablet */
@media (max-width: 768px) {
}

/* Landscape phones and down */
@media (max-width: 480px) {
}
